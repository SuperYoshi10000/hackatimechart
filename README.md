# Hackatime Chart

A website that lets you see your hackatime data, including what projects you worked on and when, and the total amount of time you worked on projects each day.



This project was made with Svelte and SvelteKit, and is running on Hack Club Nest

## Installation and Running

To install:
```sh
git clone https://github.com/SuperYoshi10000/hackatimechart
pnpm i
```
Also, copy the `.env.example` file and rename it to `.env`. No values need to be changed here.

To run:
```sh
pnpx vite --port=<port-to-run-on>
# Example: pnpx vite --port=4000
```
